    Q: 計算 1 x 2 x 3 x ... x 99 x 100 的乘法結果後面會有幾個零？共有幾位數？

    解析解:
1) 因為2x5=10，所以只要找到一對2與5，這樣就會讓乘法結果後面多一個零。
2) 觀察題目可知 2 的數量會比 5 多，因此找出所有因數 5 的個數即可。
   100/5 = 20;  先找5的倍數 <-- 一個5。
   100/25 = 4;  再找5*5=25的倍數 <-- 兩個5。
   20+4 = 24，因此計算結果後面會有24個零！
    
也可以直接寫Python程式，來算出真正的結果，接著就能夠確定後面連續出現零的個數了！
print("This is Python!")
result = 1
for k in range(1, 101):  # 1 ~ 100。 : 表示還有下文！
    result = result * k
print(result)
resultStr = str(result)  # 將整數轉換成字串！

zeros = 0  # 存零的個數。
index = 1  # 記錄指向最後面字元的位置。
while( resultStr[-index] == '0' ):  # 從後面往前檢視是否是零！
    zeros = zeros + 1
    index = index + 1

print("共有位數:", len(resultStr))
print("尾端共有幾個零:", zeros)
